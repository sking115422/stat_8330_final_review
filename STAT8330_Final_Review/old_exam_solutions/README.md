# Old Final Exam Solutions

This directory contains the original old final exam PDFs and standalone Python solution notebooks.

## Files

- `FInalExam8330_Fall2022.pdf`: original Fall 2022 old final exam.
- `Fall2022_full_solutions.ipynb`: complete Python solutions for the Fall 2022 exam.
- `FInalExam8330_Fall2023.pdf`: original Fall 2023 old final exam.
- `Fall2023_full_solutions.ipynb`: complete Python solutions for the Fall 2023 exam.

## Design

Each solution notebook is self-contained and does not import local utility files. Code is intentionally explicit and heavily commented so the patterns can be adapted during timed study.

## Exam Integrity Reminder

These notebooks are for studying old exams. Before using any notes, code, or AI-created materials during an actual exam, confirm the current course rules.
